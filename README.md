# Grit's Quest
A free and open-source game written in GameMaker.

# This game is in early alpha, here be dragons.
You play as Grit, a hungry rat looking for cheese. I made the game with inspiration from Mario and Sonic, and my favourite animal species, RATS.
Right now the game is super buggy and is missing basic features like enemies and the ever-important cheese, but I'm working on it!
